Qui sommes-nous?


Le projet Eco Transport Campus a été réalisé dans le cadre du cours de système d’informatiques d’entreprise et Cloud « SOA » en cinquième année à Polytech Paris Sud.
Le sujet consistait à mettre en place une application web de gestion des déplacements pour les étudiants de Polytech Paris Sud.
L’intérêt du projet est de développer une application permettant d’aider les étudiants, en temps réel, de gérer leurs déplacements quotidiens de la façon la plus écologique tout en permettant une bonne réactivité.

Vous aller pouvoir :

  • Vérifier en temps réel les conditions de circulation sur un trajet donnée afin d’anticiper des difficultés sur les routes.
  • Consulter l’état du Traffic RATP et bus en temps réel.
  • Chercher des annonces de possibilités de co-voiturage sur certains trajets.
  • Visualiser les statistiques concernant les trajets effectués.
  • Indiquer la possibilité de partager le véhicule si 3 personnes font un trajet commun tous les jours.

Afin de réaliser ce projet, la promo APP5 représente le seul prestataire de service.



Nous sommes une équipe de 21 apprentis ingénieurs en informatique.
Chaque groupe s’est chargé de développer un module pour le projet.
