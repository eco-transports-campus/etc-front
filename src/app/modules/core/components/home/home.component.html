Bienvenue sur l’application de gestion des déplacements pour les étudiants Polytech Paris sud


Cette application vous permet de gérer vos déplacements quotidiens de la façon la plus écologique
Vous aller pouvoir :


  • Indiquer en temps réel les conditions de circulation sur un trajet donnée afin d’anticiper des difficultés sur les routes
  • Donner l’état du Traffic RATP et bus en temps réel
  • Envoyer des annonces de possibilités de co-voiturage sur certains trajets
  • Conserver les statistiques concernant les trajets effectués
  • Indiquer la possibilité de partager le véhicule si 3 personnes font un trajet commun tous les jours
